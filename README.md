SocialHub - Full Stack Social Media Platform...

Overview: SocialHub is a full-stack social media application developed using Java Spring Boot, React, and MySQL. The platform allows users to register, authenticate securely using JWT, create posts, comment and like.

Features
*********

Authentication & Security
----------------------------------------
•	User Registration
•	User Login
•	JWT Authentication
•	Password Encryption using BCrypt
•	Protected API Endpoints
========================================

User Features
----------------------------------------
•	Create Account
•	View User Profile
•	Update Profile Information
•	Upload Profile Picture
========================================

Post Features
----------------------------------------
•	Create Posts for user
•	View All Posts Feed
•	View Single Post
•	View User Posts
•	Delete Own Posts
========================================

Social Features
----------------------------------------
•	Like Posts
•	Unlike Posts
•	Comment on Posts
•	Follow Users
•	Unfollow Users
========================================

Feed Features
----------------------------------------
•	Latest Posts Feed
•	Like Count
•	Comment Count
•	Post Author Information
========================================
•	User Interaction Status
